
```math
\text{分类属性} \triangleq \lambda x.\phi(x) \quad \text{(描述性本质)}
\text{倾向属性} \triangleq \lambda x.\Diamond\psi(x) \quad \text{(潜在可能性)}
```

| 属性类型 | 认知维度 | 示例     | 科学基础       |
| -------- | -------- | -------- | -------------- |
| 分类属性 | 静态描述 | 晶体结构 | X 射线衍射分析 |
| 倾向属性 | 动态潜能 | 化学活性 | 分子轨道理论   |
